About

This repository does not contain the full collection but those works which haven't already been covered by our CSEL digitisation effort. A Wikipedia link to this repository can be viewed here.

Contents

These XML files were produced by data entry company Jouve who OCR'd, corrected and encoded the text in accordance with the latest EpiDoc standards. While the initial results are adequate, more can and should be done to improve the XML.
